First-time setup: getting your OAuth Client ID (5 minutes, free)
  1. Go to console.cloud.google.com → create a project (call it "Empire Dashboard")
  2. APIs & Services → Library → search Google Analytics Data API → Enable
  3. APIs & Services → OAuth consent screen → External → fill in app name + your email → add yourself as a Test User
  4. APIs & Services → Credentials → Create Credentials → OAuth client ID → Web application
  5. Under "Authorized JavaScript origins" add: http://localhost:8000
  6. Copy the Client ID and paste it above
  7. Run the dashboard with a local server: open a terminal in this file's folder and run python3 -m http.server 8000 (or python -m http.server 8000 on Windows), then visit http://localhost:8000/empire-dashboard.html

Everything stays on your machine. The dashboard talks directly to Google's API with read-only Analytics access, no server, no third parties.
