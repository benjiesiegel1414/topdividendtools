First-time setup: getting your OAuth Client ID (5 minutes, free)
  1. Go to console.cloud.google.com → create a project
  2. APIs & Services → Library → search Google Analytics Data API → Enable
  3. APIs & Services → OAuth consent screen → External → add your GA email as a Test User
  4. APIs & Services → Credentials → Create Credentials → OAuth client ID → Web application
  5. Under "Authorized JavaScript origins" add the domain where this page is hosted (e.g. https://topdividendtools.com)
  6. Copy the Client ID and paste it above

Everything stays in your browser. Read-only Analytics access, no server, no third parties.
